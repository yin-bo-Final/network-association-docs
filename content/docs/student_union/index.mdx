# 软件学院学生会招新啦！
* 软件学院学生会，是学院与学生间紧密联结的桥梁纽带，更是扎根服务学生一线的核心组织，始终以“全心全意为学生服务”为根本宗旨。我们的工作围绕学生需求全方位铺展：全力配合学院深化学风建设，为大家筑牢积极向上的学习氛围；精准聚焦学生生活难题，切实解决校园日常中的各类诉求；精心策划多元活动——既有沙画创作、“一二九”大合唱、科技文化艺术节等文体科技盛宴，也有防诈骗宣讲、义务维修等暖心公益服务，既为课余生活注入鲜活色彩，更助力学生精神文明素养提升。每一场活动的落地、每一次服务的践行，都凝聚着我们“为同学做实事、为学院促发展”的初心，以实际行动守护学生权益、传递校园温情。

* 这里是磨砺成长的沃土，让你在实践中锤炼能力；这里是绽放才华的主场，让你在热爱中实现价值。我们诚挚邀请每一位怀揣热忱的你加入，让我们以同心赴新程、以协力谱新篇，在服务同学、建设学院的征途上书写属于我们的精彩，共绘软件学院的璀璨未来！期待与你在此相逢，携手开启全新旅程！

## 部门风采：

### 一．秘书处
作为学生会的中枢协调部门，主要负责以下工作：
1.	学分管理服务：负责学院成员参与活动后的第二课堂学分统一录入，确保学生学分记录准确、规范，保障学生权益； 
2.	活动宣传支撑：开展学院活动前期宣传文案撰写、现场拍摄记录及后期素材整理工作，为活动高效推进与成果留存提供有力支持； 
3.	部门运维与沟通：承担部门例会记录、物资借还管理，同时精准落实“上情下达、下情上传”，双向同步学院通知与学生诉求，保障部门运转有序、信息传递通畅。
  
图2：软件学院学生代表大会                图3：沙画创作活动
### 二．生活权益部
作为学校与学生之间的沟通桥梁，主要负责以下工作： 
1.权益安全保障：致力于维护学生正当生活权益，对校园安全隐患、生活问题予以记录并及时上报，为学生校园生活保驾护航；
2.食堂就餐建设：关注学生饮食问题，积极开展失物招领，助力营造规范、文明的食堂就餐氛围；
3.沟通桥梁搭建：搭建学生与学校间的沟通桥梁，畅通意见反馈渠道，全方位为学生解决校园生活中的各类问题。
  
图5：开学迎新活动                   图6：餐厅文明监督
### 三．文艺部
作为学院的艺术活力引擎，主要负责以下工作:
1.联络协调：及时传递活动信息，保障各项文艺活动高效推进。
2.文化建设：组织开展各种形式的文艺活动，用创意和执行力打造校园文艺盛宴。
3.平台搭建：构建开放包容的展示空间，陪伴文艺人才成长，让校园文化更有生命力。

### 四．体育部
作为学院的健康活力引领者，主要负责以下工作：
1.	赛事操盘：从学院联赛到校运动会，全程主导学院赛事策划与执行，解锁“体育活动主理人”技能；
2.	热血社交：和一群运动发烧友组队挥洒汗水，把课余时光变成充满活力的热血记忆；
3.	成长升级：在组织协调、团队协作中锤炼硬实力，让你的大学生活因体育更具质感。
  
图9：校运动会
### 五．外宣部
作为学院的对外形象窗口，主要负责以下工作：
1.宣传推广：为院系各类活动开展前期、后期全阶段宣传，传递活动信念与内容，同时监督反馈宣传效果；
2.外联拓展：积极与兄弟院校及校外公司、企业加强联系交流，为学生会和学校活动筹备资金；
3.能力提升：协助学生会其他部门开展工作，在此过程中有效提升自身策划与交际能力，为未来职场发展筑牢根基。
我们是学院的“创意引擎”与“最强外挂”！

### 六．文化科创部
作为学院的学风建设与技术服务平台，主要负责以下工作：
1. 学风建设推进：组织辩论赛、朗诵活动及各类学习讲座，营造浓厚校园学习氛围，同时关注并反馈学生学习难题，搭建师生交流桥梁，助力学院学风建设；
2. 技术服务支撑：负责学院宣传网页、宣传课件制作，完成院系活动视频特效制作，开展网络技术相关活动，同时面向全院师生提供义务维修服务，解决电脑、电子设备等常见技术故障；
3.技术记录与融合：承担院系重大活动视频收集工作，以技术赋能校园文化宣传，推动学院文化建设与科技应用深度融合，提升校园文化传播效果。

1.理想信念坚定。热爱和拥护中国共产党的领导，坚决执行党的路线、方针、政策，具有共产主义远大理想和中国特色社会主义共同理想，具有强烈的爱国意识、爱国情感。
2.道德品行端正。积极弘扬和践行社会主义核心价值观，诚实谦虚、崇尚科学、反对邪教，具有较好的群众基础；模范遵守国家法律法规，模范遵守学校的各项规章制度，无违纪行为。
3.工作作风扎实。公道正派，求真务实，具有较好的表达能力及一定的公文写作能力；作风扎实、乐于奉献，热爱团学工作，具有全心全意为同学服务的觉悟和能力。
